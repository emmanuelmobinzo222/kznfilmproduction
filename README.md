# KZN Film Production — static website

Plain **HTML**, **CSS**, and **JavaScript**. No build step or Node.js required.

## Preview

1. Open the project folder.
2. Double-click **`index.html`**, or drag it into your browser.

Ensure `index.html`, `css/styles.css`, and `js/main.js` stay in the same relative paths.

## Files

| File | Role |
|------|------|
| `index.html` | All page sections and content |
| `css/styles.css` | Layout, theme, animations |
| `js/main.js` | Mobile menu, scroll reveal, contact form demo |
